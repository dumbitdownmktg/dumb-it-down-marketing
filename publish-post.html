Publish a blog post

Paste the draft Gobii emailed you, fill in the metadata, and hit Publish. It commits directly to GitHub and Netlify deploys in ~30 seconds.

Your token is never stored — it only lives in this browser tab. Get one at: github.com → Settings → Developer settings → Personal access tokens (classic) → scope: repo

Use ## for headings, **bold** for bold, > for blockquotes, - for bullet lists. The template handles the styling automatically.

How this works
1
Gobii researches the topic, writes the post in the Dumb It Down voice, and emails the draft to [email protected]
2
You read it, make any edits, then come to this page and paste the final version into the body field above
3
Hit Publish. The script creates posts/[slug].html using the post template, then updates blog.html to include the new post in the index
4
GitHub commits. Netlify detects the commit and auto-deploys. Post is live in about 30 seconds.
5
Your GitHub token is never stored or sent anywhere except GitHub's API. It lives only in this browser tab and is cleared when you close it.
